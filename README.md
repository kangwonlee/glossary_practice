# Glossary Practice
Glossary Practice

## How to use

### Initialize a Glossary JSON file
* prepare `??.glossary` plain text files<br>
    Each line should be in `word or phrase - explanation` form
* `python merge_glossary.py [new JSON filename]`

### Practice
* `python practice.py [new JSON filename]`
